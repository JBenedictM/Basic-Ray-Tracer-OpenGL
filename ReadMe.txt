For changing scenes:
1: Go into program.cpp
2: Uncomment the lines corresponding to each scene under the start function

To change to scene 1:
	Simply uncomment the line with containing the textfile "scene1Modified.txt" under the start function in program
	
To change to scene 2:
	Simply uncomment the line with containing the textfile "scene2Modified.txt" under the start function in program
	
To change to my own scene:
	Simply uncomment the line with containing the textfile "myScene.txt" under the start function in program
